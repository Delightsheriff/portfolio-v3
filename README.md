# Delight Sheriff

Portfolio built with Next.js 16, Sanity CMS, Tailwind v4.

**Live:** [delightsheriff.com](https://delightsheriff.com)

## Quick Start

```bash
cp .env.example .env.local   # add Sanity credentials
npm install && npm run dev
```

## Commands

| Command | Purpose |
|---------|---------|
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm run lint` | Lint check |
| `npx tsc --noEmit` | Type check |

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind v4 · Sanity v5 · Framer Motion
